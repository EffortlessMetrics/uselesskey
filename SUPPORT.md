# Support

Thank you for using `uselesskey`! We want to help you get the most out of it.

## How to get help

### GitHub Issues

If you've found a bug or have a feature request, please [open an issue](https://github.com/EffortlessMetrics/uselesskey/issues).
Before opening a new issue, please search existing issues to see if someone else has already reported it.

### Discussions

For general questions, "How do I...?" queries, or sharing how you're using `uselesskey`, please use the [GitHub Discussions](https://github.com/EffortlessMetrics/uselesskey/discussions) (if enabled) or open an issue with the "question" label.

## Security

If you believe you've found a security vulnerability, please refer to our [Security Policy](SECURITY.md) for reporting instructions. Do **not** report security vulnerabilities via public GitHub issues.

## Reporting Bugs

When reporting a bug, please include:

- The version of `uselesskey` you are using.
- Your Rust version (`rustc --version`).
- Your operating system.
- A minimal reproduction if possible.
